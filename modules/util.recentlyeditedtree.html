Const nodeAdd

  • nodeAdd(tree: any, newPath: any): void
    • Defined in src/util/recentlyEditedTree.ts:87

    Manually adding node to the existing tree object by mutating. Uses special case of nodeChange.

    example

    A.B.C.D.E.F A.B.C.G.H.I

    If we want to add A.B.C.K.L ,we pass ['A','B','C','K','L'] will as newPath to nodeAdd

    Note: You should be sure that node A.B.C.K.L is not already available in the tree

    nodeAdd will just pass ['A','B','C','K','L'] as both second (oldPath) and third parameter (newPath) for nodeChange because commonPath derived from oldPath inside nodeChange will be ['A','B','C'] anyways. (This will eliminate use of DUMMY_TOKEN too)

    Parameters

    • tree: any

      Nested object representing tree.

    • newPath: any

      Array of thoughts.

    Returns void
