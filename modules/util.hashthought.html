Const hashThought

hashThought: (Anonymous function) & MemoizedFunction = _.memoize((value: string) =>globals.disableThoughtHashing ? value : _.flow([// placed before stripEmojiWithText because stripEmojiWithText partially removes angle bracketsstripTags,lower,trim,stripEmojiWithText,singularize,murmurHash3.x64.hash128,])(value))
  • Defined in src/util/hashThought.ts:46

Generate a hash of a thought with the following transformations:

  • case-insensitive
  • ignore punctuation & whitespace (when there is other text)
  • ignore emojis (when there is other text)
  • singularize
  • murmurhash to prevent large keys (Firebase limitation)

Stored keys MUST match the current hashing algorithm. Use schemaVersion to manage migrations.
